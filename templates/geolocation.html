What is this project about?

Check geolocation based on provided IP or URL address.

You can check geolocation, add it or delete records to database.

Provided data is based on api/ipstack.com

Requires authorisation via JWT.

You can interract with the API only programmatically.

Use tools like or send your own request to the endpoints provided below to get acces to the API

Endpoints:

  • /api/token

    send POST request with login credentials to get ACCESS TOKEN

  • /api/token/refresh

    send POST request with refresh token to refresh your access token

  • /api/ip-geolocation ? ip = IP_OR_URL

    get geolocation for provided IP or URL - Token required

  • /api/ip-geolocation

    list all ip geolocation data, or create a new one - Token required

  • /api/ip-geolocation/{ip or id}

    retrieve or delete geolocation data for the specified ip - Token required
